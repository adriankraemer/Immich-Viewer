## Requirements

- Apple TV (4th generation or later)
- tvOS 15.0 or later
- An Immich server that's accessible on your network

## Getting Started

1. **Launch the app** on your Apple TV
2. **Sign in** with your Immich server URL (e.g., `https://your-immich-server.com`)
3. **Enter your credentials** - either your email & password, or an Immich API key
4. **Start browsing** your photos using the Apple TV remote

## Settings

Customize your experience in the Settings menu:

- **Display**: Choose navigation style, enable/disable animations
- **Slideshow**: Adjust timing, effects, and auto-start options
- **Top Shelf**: Configure which photos appear on your home screen
- **Accounts**: Add or switch between multiple Immich accounts

## Troubleshooting

**Can't connect to server?**
- Make sure your Apple TV and Immich server are on the same network
- Verify your server URL is correct (include `https://` or `http://`)
- Check that your Immich server is running and accessible

**Authentication issues?**
- Double-check your email and password
- If using an API key, make sure it's valid and hasn't expired
- Try signing out and signing back in

**Photos not loading?**
- Check your network connection
- Ensure your Immich server has enough storage and resources
- Try restarting the app
